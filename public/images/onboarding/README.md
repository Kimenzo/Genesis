## Recommended Image Sizes
- Mascot: 400x400px (transparent PNG)
- Themes: 300x300px (transparent PNG)
- Features: 200x200px (transparent PNG)
- Roles: 200x200px (transparent PNG)
- Quiz options: 120x120px (transparent PNG)

All images should have transparent backgrounds for best results on the dark UI.
