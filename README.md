## ğŸ Getting Started

Ready to build your empire?

1. **Clone the Repository**
   ```bash
   git clone https://github.com/your-repo/genesis.git
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment**

   ```env
   VITE_SUPABASE_URL=your_url

   ```

   ```bash
   npm run dev
   ```
